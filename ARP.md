# ARP (Address Resolution Protocol)

## What is ARP?
ARP resolves IP addresses to MAC addresses on a local network.

## Use in Wireshark
- ARP Requests: "Who has `<IP>`?"
- ARP Replies: "`<IP>` is at `<MAC>`"

## Wireshark Filter
- `arp`: Displays all ARP traffic.
